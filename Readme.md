## 遥控使用：
- 新镖架的最新代码标签：`26镖架_单发无控镖_发射成功`中，全系统校准只会在上电后进行一次，而滑块电机还会在每次发射流程中下拉前进行校准。
- 如果出现yaw联轴器松动或者行程电机联轴器松动导致位置偏移，只能按reset键进行全系统校准。
- 遥控断连后，只会进入电机失能保护。不再触发重新校准。
- 取消右摇杆手动切换yaw和行程电机控制状态的功能，转而变成自动切换。即默认手动拨R摇杆控制yaw和行程，如果调参板接入，则后续都采用调参板数据。如果视觉接入，后续都采用视觉数据，并且优先级高于调参板。不过目前是一次性切换，即优先级高的介入后就无法回退。所有控制模式均包含软件限位，不用担心参数意外给大了。
- 增加紧急预案功能，由于经常出现扳机舵机拉不动导致发射失败，滑块卡在扳机处无法释放的情况，故增加手动安全释放的功能：
	- 当发射流程暂停时，例如S1=middle或者一次发射结束停下，如果S2拨到底进入紧急预案，此时滑块电机被右摇杆手动接管，操作者可以通过手动把滑块拉到底让电机拉住蓄力的滑台，然后手动松开扳机后，再拨RY摇杆慢慢把滑块放上缓冲区，实现安全释放。
	- 需要注意，紧急预案需要电机使能，所以s1不能拨到up。

- 简洁版使用说明：
    - S1拨杆：up-失能，middle-暂停，down-自动发射
    - S2拨杆：up-无，middle-装填升降机测试，down-装填出仓加升降机测试+紧急预案开启
    - 调参板接入后，优先级高于摇杆控制yaw和滑块电机。
    - RX摇杆：控制yaw目标角度
    - RY摇杆：控制蓄力行程电机/滑块目标位置
